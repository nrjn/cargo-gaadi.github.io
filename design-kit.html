A workshop palette, not a coffee shop.

14 · Rules of consistency

If you find yourself reaching for these, stop — the design system already handles it.

Do
  • ✓ Use var(--c-forest) for every primary CTA
  • ✓ Use var(--c-signal) only on "ours" markers + key highlights
  • ✓ Alternate paper → paper-2 → dark for section rhythm
  • ✓ Pick from the spacing scale always
  • ✓ One CTA per section, primary action only
Don't
  • ✗ Add a new color "just for this section"
  • ✗ Use pure black #000 or pure white #fff
  • ✗ Hardcode color hexes in HTML — always via tokens
  • ✗ Use signal yellow as a background fill on large areas
  • ✗ Use terracotta / cream — these were the Anthropic palette and are out
